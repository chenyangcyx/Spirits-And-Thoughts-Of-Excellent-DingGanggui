该脚本针对“优秀的丁港归同学精神思想”进行部署
作用：
1、将该项目自动的部署到Linux服务器的目录中
2、定时更新项目内容，自动从git地址pull最新文件

文件说明：
1、clone.sh实现第1项功能
2、pull.sh实现第2项功能
